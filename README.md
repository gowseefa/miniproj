# 🏥 Meditrack AI  
### Intelligent Healthcare Management and Diagnosis System

---

## 📌 Overview
Meditrack AI is a full-stack healthcare management web application designed for doctors. It helps manage patients, staff, and medical reports while integrating AI-based diagnosis using symptoms and X-ray analysis.

---

## 🚀 Features

### 🔐 Authentication
- Secure doctor login system
- Session-based access control

### 👨‍⚕️ Doctor Dashboard
- View total staff
- Add / remove staff members
- Manage system access

### 🧑‍🤝‍🧑 Patient Management
- Add new patients
- View patient records
- Delete patient data

### 📊 Reports Module
- Upload and manage medical reports
- Store patient diagnosis details

### 🤖 AI-Powered Features
- 🩻 X-ray Analysis  
  Upload X-ray images and get AI-generated explanations  
- 💊 Medicine Suggestions  
  Suggests medicines based on X-ray and symptoms  
- 🧠 Symptom Checker  
  Enter symptoms and get possible conditions + suggestions  

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Jinja2  
- **Database:** SQLite  
- **AI Integration:** Google Generative AI
